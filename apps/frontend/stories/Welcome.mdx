## Project Context

**Reiseklar** (Norwegian for "Travel Ready") is a journey planning application that:
- Integrates with Entur API for Norwegian public transport
- Provides real-time weather information
- Supports user authentication and profiles
- Offers admin analytics dashboard
- Enables saving favorite routes and trips
